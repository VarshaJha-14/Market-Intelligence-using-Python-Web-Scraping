# 🌐 HTML Parsing Playground

> A collection of Python exercises demonstrating HTML parsing, DOM navigation, and data extraction using BeautifulSoup.

---

# 📖 Overview

Before extracting useful information from websites, it's important to understand how HTML documents are structured.

This repository explores the fundamentals of HTML parsing using BeautifulSoup and demonstrates different techniques for navigating the Document Object Model (DOM) and extracting structured information from webpages.

The exercises in this project laid the foundation for my later web scraping projects involving book catalogues, IMDb movie data, and quote extraction.

---

# 🎯 Objective

Learn how to:

- Parse HTML documents
- Navigate the DOM
- Locate HTML elements
- Extract text and hyperlinks
- Traverse parent-child relationships
- Understand how BeautifulSoup represents webpages

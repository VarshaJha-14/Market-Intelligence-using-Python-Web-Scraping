# 📖 Overview

Businesses increasingly rely on publicly available web data to understand markets, competitors, customers, and trends.

This repository showcases a collection of Python web scraping projects built to automate data collection from websites and convert unstructured HTML into structured datasets suitable for analysis.

The projects demonstrate core concepts including:

- HTML Parsing
- DOM Navigation
- Multi-page Crawling
- Data Cleaning
- Browser Automation
- Metadata Extraction
- Dataset Generation

These projects represent the beginning of my journey into Python automation and later became the foundation for my work in SQL, Tableau, data analytics, and product-focused projects.

# 📂 Projects Included

## 📚 Book Store Market Analysis

**Objective**

Automate the extraction of product information from an online bookstore.

**Highlights**

- Product titles
- Prices
- Stock availability
- Categories
- Product URLs
- CSV generation

---

## 🎬 IMDb Movie Analytics

**Objective**

Collect movie information from IMDb and perform exploratory analysis.

**Highlights**

- Movie titles
- Genres
- Runtime analysis
- Most voted movies
- Year-wise insights

---

## 💬 Quote Knowledge Base

**Objective**

Create a structured database of quotes, authors, biographies, and tags.

**Highlights**

- Quote extraction
- Author information
- Birth dates
- Multi-page crawling
- Tag filtering

---

## 🖼 Wikipedia Image Analytics

**Objective**

Extract image metadata and analyze image dimensions from Wikipedia pages.

**Highlights**

- Image URLs
- Height & Width
- Image area calculation
- Largest image detection

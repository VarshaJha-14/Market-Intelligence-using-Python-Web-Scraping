# 🎬 IMDb Movie Analytics using Python & BeautifulSoup

> A Python-based web scraping project that extracts movie information from IMDb and performs basic exploratory analysis on popularity, genres, and runtime.

---

# 📖 Overview

Movie databases contain valuable information that can be used for market research, trend analysis, and audience insights.

This project demonstrates how Python and BeautifulSoup can be used to automatically collect movie data from IMDb and answer analytical questions using scraped data.

The project focuses on extracting structured movie information and transforming it into insights that can support entertainment analytics and decision-making.

---

# 🎯 Business Problem

Suppose you're a Data Analyst working for a movie streaming platform.

Your team wants to answer questions like:

- Which movies received the highest audience engagement?
- Which genres dominate the most popular movies?
- Which movie had the longest runtime?
- Which movie received the highest number of votes each year?

Instead of manually collecting this information, this project automates the process using web scraping.

---

# 🚀 Features

✅ Extracts movie titles

✅ Extracts genres

✅ Identifies highest-voted movies

✅ Finds longest runtime movie

# 📊 Analysis Performed

### 🎥 Movie Titles & Genres

Scrapes movie titles together with their genres from IMDb.

Example:

| Movie | Genre |
|--------|---------|
| Avengers: Infinity War | Action, Adventure |
| Black Panther | Action, Sci-Fi |

---

### 🏆 Most Voted Movie (2010–2014)

Automatically retrieves the highest-voted movie for each year.

Example Questions

- Which movie received the highest number of IMDb votes in 2012?
- Which movie topped audience popularity in 2014?

---

### ⏱ Longest Runtime Analysis

Among the Top 250 most-voted movies in 2018, determine which movie has the maximum runtime.
